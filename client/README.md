# TECH TREND VISUALIZER 

Developed with React Redux

## 開発環境
- react-redux based on https://github.com/KaiHotz/React-Redux-Simple-Starter  
- node v10.16.0
- yarn

## インストール
```
$ cd client
$ yarn install
```

## 開発手順
```
$ cd client
$ yarn start
```

## ビルド
```
$ cd client
$ yarn build
```

## フォルダ構成
- client/public
  - html,画像データ等
- clirnt/src
  - /actions - Redux Action
  - /assets - SVGデータ
  - /components - 各コンポーネント
  - /containers - メインページ
  - /data - Fallback用json
  - /reducers - Redux Reducer
  - /styles - main scss
